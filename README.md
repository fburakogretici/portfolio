# Burak Öğretici - Personal Portfolio

A modern, highly professional, and visionary personal portfolio landing page for a Full Stack Developer & Software Specialist. Featuring a sleek dark mode theme, interactive animations, and a fully responsive design.

🔗 **Live Link:** [https://github.com/fburakogretici/portfolio](https://github.com/fburakogretici/portfolio)

---

## ✨ Design & Key Features

- **Hero Section:** High-impact introduction with a dynamic neon glow hover effect on primary Call-to-Action buttons and a subtle, lightweight, tech-inspired animated grid/particle background.
- **About & Expertise:** Detailed showcase of professional experience in software architecture, large-scale systems (such as Hospital Information Management Systems - HBYS), and complex API integrations. Includes a responsive tech stack grid grouped into:
  - *Backend & Database:* .NET, C#, Node.js, SQL Server, Supabase
  - *Frontend:* React, Next.js
  - *Automation & AI:* n8n, Tally, Generative AI API integrations
- **Featured Projects:** Interactively animated project cards with lift and border glow effects showcasing major projects:
  - **NoCodePage:** A modern website builder and SaaS platform.
  - **Aurox Software:** Digital operational process solutions for businesses.
  - **Autonomous Content Engine:** Zero-human-intervention content production system built with n8n and GenAI.
- **Contact Form:** A minimalist, fully functional contact form with built-in input icon design and validation UX.
- **Footer:** Clean layout featuring copyright details, animated heartbeat indicator, and social connections (LinkedIn & GitHub).
- **Responsive Layout:** Tailored with CSS grid, flexbox, and media queries to look premium on mobile, tablet, and desktop devices.
- **Scroll Animations:** Smooth scroll-reveal effect powered by `IntersectionObserver`.

---

## 🛠️ Tech Stack & Libraries

- **Markup:** HTML5 (Semantic Structure)
- **Styling:** Vanilla CSS (Modern CSS variables, flexbox, grid, glassmorphism, CSS keyframe animations)
- **Logic & Interactions:** Vanilla JavaScript (ES6+, Intersection Observer API)
- **Icons:** FontAwesome / SVG Icons

## 🚀 How to Run Locally

Since this is a lightweight, high-performance static website, you don't need any complex installation steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/fburakogretici/portfolio.git
   cd portfolio
   ```
2. **Open in Browser:**
   - Double-click `index.html` in your file explorer to open it instantly.
   - Or, if you use VS Code, right-click `index.html` and choose **"Open with Live Server"** for hot-reload support.
